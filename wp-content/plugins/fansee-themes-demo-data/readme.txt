== Description ==

Import your demo content, widgets and theme settings with one click for [Fansee Themes](https://www.fanseethemes.com/) official themes.

This plugin uses One Click Demo Import plugin By [Proteus Themes] (https://wordpress.org/plugins/one-click-demo-import/) to import and process data.

Also we are using a service hosted on our [server] (https://fanseethemes.com) to fetch latest demo data.
https://www.fanseethemes.com/terms-and-conditions/

Get free support at [https://www.fanseethemes.com/]((https://www.fanseethemes.com/))

== Installation ==

1. Upload the `fansee-themes-demo-data` folder to your `/wp-content/plugins/` directory or alternatively upload the fansee-themes-demo-data.x.x.x.zip file via the plugin page of WordPress by clicking 'Add New' and selecting the zip from your computer.
2. Activate the Fansee Themes Demo Data WordPress plugin through the 'Plugins' menu in WordPress.
4. Goto Appearance / Import Demo Data.
